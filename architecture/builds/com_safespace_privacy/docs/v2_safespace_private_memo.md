## 🎨 1. 물리 엔진 기반 Obsidian 스타일 마인드 그래프

Phase 2에서 가장 집중한 과제는 단순한 텍스트 리스트 형태를 뛰어넘어, 사용자가 메모 간의 연관성과 태그 클러스터를 유기적으로 탐색할 수 있는 **자체 물리 법칙 시뮬레이션 기반 마인드 그래프**를 순수 Dart의 `CustomPainter` 상에 정밀 설계한 것입니다.

### ⚙️ 핵심 물리 공식 및 설계 원칙
* **쿨롱의 법칙(Coulomb Repulsion) 기반 노드 간 척력 ($F_r$)**:
  임의의 노드 쌍 $(i, j)$ 간의 거리 $d$에 대해 서로를 밀어내는 힘을 인가하여 노드가 서로 겹치거나 뭉치지 않게 방어합니다.
  $$F_r = \frac{K_{repulsion}}{d^2} \quad (K_{repulsion} = 1400)$$
* **훅의 법칙(Hooke's Spring Law) 기반 노드 간 탄성 인력 ($F_a$)**:
  메모와 태그 노드가 결합되어 있거나 동일한 카테고리를 가질 경우, 팽팽한 스프링 끈으로 연결된 것 같은 인력을 작용시킵니다.
  $$F_a = k_{spring} \times (d - d_{natural}) \quad (k_{spring} = 0.05, \, d_{natural} = 100)$$
* **속도 감쇠(Damping Coefficient)**:
  물리 연산 루프가 무한히 흔들리며 요동치지 않고, 매 프레임마다 이전 속도의 `85%`로 서서히 마찰 제동을 주어 우아하게 수렴하도록 구성했습니다.
* **제스처 및 클러스터 인터랙션**:
  두 손가락을 활용한 스케일(Pinch Zoom), 팬(Pan) 조작을 완벽하게 지원하며 특정 태그 노드 탭 시 인접 클러스터를 제외한 전체 화면을 15% 불투명도로 페이드아웃하는 연출을 적용했습니다.

## 📝 2. Notion 스타일 마크다운 토글 파서 & 체크리스트

외부 의존 패키지를 일절 사용하지 않고, 렌더링 성능과 모바일/웹 크로스 플랫폼 빌드 안전성을 극대화하기 위해 직접 마크다운 빌딩 블록 파서를 제작했습니다.

* **인터랙티브 체크박스 뷰어**:
  문서 내 `- [ ] 할 일` 혹은 `- [x] 완료` 텍스트 라인을 식별하여 터치 가능한 실시간 체크박스 위젯으로 렌더링합니다.
* **실시간 DB 양방향 동기화**:
  사용자가 체크박스를 클릭하면, 시스템은 문서의 해당 라인을 치환하여 데이터베이스(`Shared Preferences`)의 마크다운 원본 텍스트를 실시간으로 재작성합니다. 이때 햅틱 피드백을 전달하여 프리미엄 조작감을 완성시켰습니다.
* **텍스트 빌딩 블록화**:
  헤더 파싱(`#`, `##`), 코드 블록 백틱(```) 및 클립보드 원클릭 복사 시스템, 그리고 굵은 글씨(`**굵게**`) 파싱을 모듈화하여 지원합니다.

## 🔒 3. 생체 PIN 장벽 및 Sentinel 보안 감시 체계

온디바이스 프라이버시 원칙에 근거하여 로컬의 시크릿 카테고리(`Vault`) 접근 시 강력한 잠금 모듈을 가동합니다.

* **마스터 PIN 인증 배리어**:
  잠금 처리된 비밀 메모 카드에 접근할 시 마스터 PIN(`2026`)을 요구하며, 성공하지 못하면 모든 텍스트가 `•••••••••` 처리되어 메모 유출을 원천 방어합니다.
* **Sentinel 오프라인 탐지 로그**:
  성공적인 보안 쉴드 우회(`SUCCESS`) 및 불법 혹은 잘못된 핀 입력 시도(`WARNING`) 내역이 로컬 기기 내부에 실시간 타임라인으로 보존되어 침입 추적 대시보드를 이룹니다.

## 🛠️ 4. 빌드 결과 및 릴리즈 호환성 검증

정적 무오류 및 크로스 플랫폼 최적 컴파일을 완료했습니다.

3. **배포 자동화**: 컴파일된 바이너리를 `active_web_preview` 심볼릭 링크로 배포하고, http://localhost:8502 포트를 연동하여 스트림릿 제어 수트(8501)와 유기적인 실시간 프리뷰를 유지하고 있습니다.
