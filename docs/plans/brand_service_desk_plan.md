# 🖥️ Solve-for-X (SFX) Brand Web Service Desk (Concept & Architecture Plan)

본 문서는 브랜드 컨트롤 타워인 `@brand-web` (Port 3000, Next.js) 내에 통합 구축할 **'지능형 통합 서비스 데스크 (SFX Intelligent Service Desk)'**의 기획 및 기술 설계안입니다. 

기존의 텔레그램 메신저 기반 대응 채널을 웹 대시보드와 유기적으로 양방향 통합(Two-way Sync)하여, 1인 관리자인 지훈님의 운영 효율성을 극대화합니다.

---

## 🎨 1. Service Desk Core Modules (두 가지 핵심 영역)

서비스 데스크는 **[공개형 고객 지원 센터]**와 **[비공개 SRE 어드민 콘솔]**의 듀얼 아키텍처로 설계되어 `@brand-web` 내부에서 완벽히 융합됩니다.

```

                                         │
                 ┌───────────────────────┴───────────────────────┐
                 ▼                                               ▼
     [공개형 고객 지원 센터]                               [비공개 SRE 어드민 콘솔]

   • 일반 고객 문의 즉시 접수                             • 실시간 수집 티켓 파이프라인 관제
   • 실시간 AI 자동 해결책 추천                            • AI 답장 제안 편집 & 전송 승인 관문
   • Neon 감성 맞춤형 FAQ 아카이브                         • 유저 SSO & 서버 로그 즉각 크로스 진단
```

## 🔬 2. Key High-Value Features (브랜드 가치를 10배 높일 차별성)

### 2.1. Instant Diagnostics Synergy (로그 크로스 매핑)
*   일반적인 고객센터 솔루션(Zendesk 등)은 문의 텍스트만 보여주지만, SFX 서비스 데스크는 데이터베이스의 **SSO 계정 고유 ID (UUID)**와 동기화됩니다.
*   고객이 "Memento Mori 그리드가 해금되지 않아요"라고 접수하면, 어드민 콘솔은 **해당 고객의 실제 4,160주 라이브 그리드 데이터와 최근 API 트래픽 로그를 티켓 옆에 즉시 팝업**해 줍니다. 
*   지훈님은 단 3초 만에 오류 코드를 직관적으로 대조 및 파악할 수 있습니다.

### 2.2. Interactive AI Draft Editor (승인 전용 웹 편집기)
*   텔레그램 Inline 버튼을 통한 1-Click 승인뿐만 아니라, **더 세밀하고 정중한 답장 편집이 필요할 때** 어드민 서비스 데스크의 에디터를 활용합니다.
*   AI가 작성해 놓은 초안을 마크다운 편집기에서 2~3글자 수정하여 [즉시 발송] 버튼을 누르면, **Gmail API를 통해 고객 메일로 회신이 즉시 전송**됩니다.

### 2.3. Public Neon Support Center (고객이 반하는 다이나믹 서포트 페이지)
*   글래스모피즘과 네온 테마가 이식된 프리미엄 서포트 페이지를 제공합니다.
*   문의글 작성 단계에서 **유저가 에러 내용을 타이핑하는 즉시 AI가 백그라운드에서 실시간 매칭 FAQ를 부드러운 애니메이션으로 제안**하여 실제 티켓 생성률을 사전 방어(Deflection)합니다.

## 🚀 4. Expected Impact & Value (도입 시 기대 효과)

1. **브랜드 신뢰도의 극대화:** 유저가 메일이나 스토어에 에러 글을 남겼을 때, **10분 내에 정확한 시스템 오류 진단과 함께 지훈님의 승인이 완료된 정교한 기술 답변**을 수신하게 되므로 압도적인 프리미엄 사후 지원 경험을 선사합니다.
2. **완벽한 통합 운영 허브:** `@brand-web`이 단순 소개 페이지를 넘어, **진정한 SaaS형 SRE 통제실이자 고객 커뮤니케이션 허브**로 완성됩니다.
3. **독립성 확보:** 외부 유료 CS 툴(Zendesk, Channel.io)에 매월 10~20만 원 상당의 비용을 지불하지 않고, **단일 데이터베이스 하에서 영구 무료 및 완전 자율형(Self-Hosted) 통제망**을 소유하게 됩니다.
